Hi! This is Habits, a Habit tracker I built for Hack Club: The Game. The app can add custom habits which you wish to follow daily, and keep daily completion streaks.

It was built using Python's CustomTkinter Library.

To use it, simply download the release and run the "main.exe" file. 

I plan to add more to this project in the future.
To contribute, go ahead and fork the repo! I'm happy to look at any pull requests.
